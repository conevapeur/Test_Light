


Le script COLORED HEADERS va vous permettre de mettre de la couleur dans vos hierarchies de GameObjects!

Pour l'utiliser, rien de plus simple, il suffit de se rendre sur le scriptableObject présent dans le dossier.

Ajoutez une entrée à la liste (avec le petit +), mettez dans "name" un mot contenu dans le nom de votre objet, choisissez une couleur 
(/!\ pensez à mettre l'alpha à 1, par défaut elle est à 0 !!!)

Devant le nom de votre game object dans votre hierarchie, ajoutez "--- ", par exemple "--- GameManager" ou "--- LEVEL" etc.

Et Tada!


ATTENTION :
A la première ouverture d'un projet utilisant ce script, ou bien la première fois que vous le mettez en place dans une scène,
vous allez avoir des messages d'erreur dans la console
et votre hierarchie ne s'affichera pas en entier !
Restez attentifs à cela. Il suffit alors d'ouvrir un script, ajouter un Enter ou un espace, Ctrl-S, Unity va recompiler et voilà, tout marche.


Have fun et mettez de la couleur dans votre vie !
