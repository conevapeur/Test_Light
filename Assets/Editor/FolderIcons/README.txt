Un véritable petit génie a laché ce script sur GitHub !

Il va vous permettre de mettre non seulement de la couleur dans vos dossiers d'assets, mais même d'entièrement customiser vos icones de dossier

Pour l'utiliser, rien de plus simple. Rendez vous sur le scriptable object présent dans le dossier.

Ajoutez un nouveau dossier customizé dans la liste avec le petit "+"
Dans la variable "Folder" faites glisser un dossier depuis votre hierarchie d'assets 
dans Folder Icon, selectionnez une icône de dossier (le dossier arc-en-ciel c'est bonus de ma part)
Dans OverlayIcon sélectionnez une icone qui s'affichera par dessus celle du dossier.

Et voilà!

N'hésitez pas à vous faire un petit dossier d'icônes (vous pouvez en trouver pleins sur https://game-icons.net/, les customizer et les avoir sur fond transparent)
Vous pouvez également faire vos propres visuels de dossiers (comme pour le dossier arc-en-ciel)


Have fun et ajoutez de la couleur dans votre vie !
